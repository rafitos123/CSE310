# Overview


This program generates a QR code and redirects to the Church of Jesus Christ website.

This program aims to facilitate easy access to the website, simply by scanning the QR code.


[Software Demo Video](https://youtu.be/0RNFn9bMPKg)

# Development Environment

I used VS Code application to code this software.

I used C# as the programming language and the QRCoder library to generate the QR code more quickly.
